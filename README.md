# StudyWithJL

Este é um gerenciador de estudos.
Um programa que vai nos ajudar a montarmos um plano de estudo para uso pessoal com base na sua própria realidade.

O programa deve pedir ao usuário a ser cadastrado um limite de horas de estudo por dia e quantos dias da semana vai estudar, além de
informar quais são esses dias. O usuário vai cadastrar, com base no dia informado pelo sistema, qual cadeira ou disciplina vai estudar,
quais materiais vai usar e, ainda, deverá obter um acréscimo de horas estudadas a cada hora por disciplina e assim vai aumentar a sua aura:
ki, nen ou chakra com base neles, para ser encorajado a se tornar cada vez "mais forte" nos estudos, como Goku, ou chegar ao nível de um
verdadeiro Hokage, ou mesmo um Caçador. Futuramente, teremos as escolhas desses modos ou a mistura deles para uma possível batalha ninja,
de sayajins ou caçadores com seus respectivos poderes. A meta é fazer um sistema de estudo divertido.
